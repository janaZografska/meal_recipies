
# Meal Recipes App

A Flutter application that displays meal recipes from around the world using the TheMealDB API.

## Features

- Browse meal categories
- Search categories
- View meals by category
- View complete meal details with ingredients and instructions
- Get random meal suggestions

### Prerequisites

- Flutter SDK (3.9.2 or higher)
- Dart SDK
- Git

### Installation

1. Clone the repository
\`\`\`bash
git clone https://github.com/janaZografska/meal_recipies.git
cd meal_recipies
\`\`\`

2. Install dependencies
\`\`\`bash
flutter pub get
\`\`\`

3. Run the app
\`\`\`bash
flutter run
\`\`\`

## API

This app uses the free [TheMealDB API](https://www.themealdb.com/api.php)

- flutter
